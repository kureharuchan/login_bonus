# Login Bonus Datapack

まだまだ作りかけです。
マインクラフト用のログインボーナスデータパックです。

## ✨ 機能

* 毎日ログインボーナス
* 累計7日ごとのガチャ報酬
* レアリティ付き報酬（ネザライト・ダイヤなど）

## 📦 導入方法

1. datapacksフォルダに入れる
2. /reload を実行

## 😉 使用について

* 報酬や確率など、カスタマイズOK！

## ⚠️ 注意

* 初回はscoreboardが自動生成されます
* マルチプレイでも使用可能です
