# Login Bonus Datapack
大体完成！
マインクラフト用のログインボーナスデータパックです。

プレイヤーごとにクールタイム制でボーナスを配布します。
例）1日目ボーナス受け取り→24時間後→2日目ボーナス受け取り

## ✨ 機能

* 毎日ログインボーナス！
* 累計7日ごとのガチャ報酬！段々ティアが上がる！
* 30日、50日、100日には特別なボーナスも！

## 📦 導入方法

1. datapacksフォルダに入れる
2. /reload を実行

## 😉 使用について

* 報酬や確率など、カスタマイズOK！

## ⚠️ 注意

* 初回はscoreboardが自動生成されます
* マルチプレイでも使用可能です
